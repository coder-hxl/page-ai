## 安装

1. 克隆此存储库：

```bash
git clone https://github.com/coder-hxl/page-ai.git
```

2. 打开谷歌浏览器，在地址栏输入 `chrome://extensions/`
3. 开启在页面右上角的 `开发者模式`
4. 点击页面左上角的加载已解压的扩展程序，选择 page-ai 的 `packages\chrome` 文件夹
